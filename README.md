Roleplay Studio (Project-HSA)</h1>
<p style="font-size: 1.2em; color: #8b949e;"><strong>당신만의 AI 배우와 함께, 살아 숨 쉬는 이야기를 만들어보세요.</strong></p>

<div style="background-color: #161b22; border-left: 4px solid #8A2BE2; padding: 25px; border-radius: 6px; margin: 30px 0;">
<p style="margin: 0; font-size: 1.1em; line-height: 1.7;">
<strong>Roleplay Studio</strong>는 Google Gemini AI를 기반으로 구동되는 본격 AI 페르소나 액팅 시뮬레이터입니다.
</p>
<p style="margin-top: 1em; font-size: 1.1em; line-height: 1.7;">
사용자는 AI가 연기하는 페르소나와 실시간으로 상호작용하며 예측 불가능한 드라마를 써 내려갈 수 있습니다. 이 프로젝트는 단순한 챗봇을 넘어, AI 스스로 능동적으로 사고하고 반응하며 사용자와의 관계 속에서 성장하는, 살아있는 인격체를 만드는 것을 목표로 합니다.
</p>
</div>

<h2 style="color: #58a6ff; border-bottom: 1px solid #30363d; padding-bottom: 10px; margin-top: 40px; margin-bottom: 20px;">✨ 핵심 기능 (Features)</h2>
<div style="padding-left: 20px; line-height: 1.8;">
<h3 style="color: #c9d1d9; margin-top: 30px;">🗼 1. 컨트롤 타워 (Control Tower)</h3>
<p>이야기의 모든 것을 지휘하는 중앙 통제실입니다. 이곳에서 당신은 장면의 모든 요소를 세밀하게 조율할 수 있습니다.</p>
<ul>
<li><strong>인물 관리:</strong> 세상에 단 하나뿐인 당신만의 페르소나를 창조하고, 템플릿으로 저장하여 언제든 재사용할 수 있습니다.</li>
<li><strong>상황 설정:</strong> 장면의 배경, 시간, 날씨, 그리고 세계관의 규칙(Lore)을 설정하여 이야기의 무대를 만듭니다.</li>
<li><strong>AI 설정:</strong> '소설/채팅' 연출 모드를 선택하고, Gemini 모델의 파라미터를 조절하여 AI의 연기 스타일을 제어합니다.</li>
<li><strong>OOC 연출:</strong> 작가의 시점에서 AI에게 직접 연기 지시를 내리거나, AI 조수 'PD 한세아'와 대화하며 영감을 얻습니다.</li>
</ul>
<h3 style="color: #c9d1d9; margin-top: 30px;">📖 2. 페르소나 시트 (Persona Sheet)</h3>
<p>이 앱의 심장. 이곳에서 당신은 페르소나의 영혼을 조각합니다.</p>
<ul>
<li><strong>다층적 프로필:</strong> 기본 정보는 물론, 페르소나의 인생을 관통하는 서사(결정적 경험, 핵심 원칙), 내면(감정 프로필, 리비도), 생활(라이프스타일, 관계망)까지 수십 가지 항목을 설정하여 입체적인 인물을 만듭니다.</li>
<li><strong>AI 기반 자동 생성:</strong> 단 몇 개의 키워드만으로 이 모든 복잡한 프로필을 AI가 자동으로 생성하게 할 수도 있습니다.</li>
<li><strong>데이터 내보내기:</strong> 완성된 페르소나 프로필을 모든 섹션이 정리된 .txt 파일로 내보내 백업하거나 다른 사람과 공유할 수 있습니다.</li>
</ul>
<h3 style="color: #c9d1d9; margin-top: 30px;">💬 3. 메인 뷰 (Main View)</h3>
<p>당신과 페르소나의 이야기가 펼쳐지는 무대입니다.</p>
<ul>
<li><strong>실시간 상호작용:</strong> 당신의 행동과 대사에 AI 페르소나가 실시간으로 반응하며, 한 치 앞을 알 수 없는 스토리를 만들어갑니다.</li>
<li><strong>AI 응답 제어:</strong> AI의 연기가 마음에 들지 않는다면, 언제든 '다시 연기'를 시키거나 '장면 계속'을 통해 다른 방향으로 이야기를 이끌어갈 수 있습니다.</li>
</ul>
<h3 style="color: #c9d1d9; margin-top: 30px;">💡 4. PD 한세아 & 아이디어 보드</h3>
<p>창작의 고통을 덜어줄 당신만의 유능한 파트너입니다.</p>
<ul>
<li><strong>PD와 브레인스토밍:</strong> 페르소나, 플롯, 설정 등 막히는 부분이 있다면 언제든 PD를 호출해 아이디어를 얻으세요.</li>
<li><strong>HTML 시각화:</strong> PD는 단순 텍스트를 넘어, 프로필 카드나 아이템 정보 등을 HTML로 시각화하여 보여줄 수 있습니다.</li>
<li><strong>아이디어 고정:</strong> 대화 중 떠오른 번뜩이는 아이디어나 마음에 드는 문장을 '아이디어 보드'에 고정하여 체계적으로 관리할 수 있습니다.</li>
</ul>
<h3 style="color: #c9d1d9; margin-top: 30px;">🖥️ 5. 실시간 페르소나 현황 창</h3>
<p>연기 중인 페르소나의 내면을 실시간으로 훔쳐보세요.</p>
<ul>
<li><strong>독립된 모니터링 창:</strong> 특정 페르소나의 '현황 보기'를 클릭하면, 별도의 창이 떠서 그의 실시간 감정 상태와 단기 목표를 보여줍니다.</li>
<li><strong>자유로운 이동:</strong> 이 현황 창은 화면 어디든 자유롭게 드래그하여 옮길 수 있어, 메인 스토리를 방해하지 않으면서 페르소나의 심리를 계속 추적할 수 있습니다.</li>
</ul>

<h2 style="color: #58a6ff; border-bottom: 1px solid #30363d; padding-bottom: 10px; margin-top: 40px; margin-bottom: 20px;">🚀 시작하기 (Getting Started)</h2>
<p>이 프로젝트를 당신의 로컬 환경에서 실행하기 위해서는 몇 가지 설정이 필요합니다.</p>
<h3 style="color: #c9d1d9; margin-top: 30px;">1. 프로젝트 복제 (Clone)</h3>
<pre style="background-color: #161b22; padding: 16px; border-radius: 6px; overflow-x: auto; font-family: 'SFMono-Regular', Consolas, 'Liberation Mono', Menlo, Courier, monospace;"><code>git clone https://github.com/your-username/roleplay-studio.git
cd roleplay-studio</code></pre>
<h3 style="color: #c9d1d9; margin-top: 30px;">2. 의존성 설치 (Install Dependencies)</h3>
<pre style="background-color: #161b22; padding: 16px; border-radius: 6px; overflow-x: auto; font-family: 'SFMono-Regular', Consolas, 'Liberation Mono', Menlo, Courier, monospace;"><code>npm install</code></pre>
<h3 style="color: #c9d1d9; margin-top: 30px;">3. 환경 변수 설정 (.env)</h3>
<p>프로젝트의 루트 디렉토리에 있는 <code>.env.example</code> 파일의 복사본을 만들어 <code>.env</code> 라는 이름으로 저장하세요.<br>그리고 아래 설명에 따라 파일 안의 내용을 자신의 키로 채워주세요.</p>
<pre style="background-color: #161b22; padding: 16px; border-radius: 6px; overflow-x: auto; font-family: 'SFMono-Regular', Consolas, 'Liberation Mono', Menlo, Courier, monospace;"><code>cp .env.example .env</code></pre><br>
<ul style="padding-left: 20px; line-height: 1.8;">
<li><strong>Firebase:</strong> Firestore 데이터베이스를 사용하여 장면과 캐릭터 데이터를 저장합니다.<ul style="padding-left: 20px;"><br>
<li><a href="https://console.firebase.google.com/" style="color: #58a6ff;">Firebase 콘솔</a>에서 새 프로젝트를 생성하세요.</li><br>
<li>프로젝트 설정 > 일반 탭에서 웹 앱(<code>&lt;/&gt;</code>)을 등록하고, <code>firebaseConfig</code> 객체에 있는 값들을<br><code>.env</code> 파일의 <code>REACT_APP_FIREBASE_*</code> 변수에 맞게 복사해 넣으세요.</li><br>
<li>Cloud Firestore 데이터베이스를 생성하고 시작하세요. (테스트 모드로 시작해도 무방합니다)</li></ul></li><br>
<li><strong>Google AI (Gemini):</strong> AI 모델을 구동하기 위한 API 키입니다.<ul style="padding-left: 20px;"><br>
<li><a href="https://aistudio.google.com/app/apikey" style="color: #58a6ff;">Google AI Studio</a>로 이동하여 API 키를 발급받으세요.</li><br>
<li>발급받은 키를 <code>.env</code> 파일의 <code>REACT_APP_GEMINI_API_KEY</code> 값으로 입력하세요.</li></ul></li>
</ul>
<h3 style="color: #c9d1d9; margin-top: 30px;">4. 애플리케이션 실행</h3>
<pre style="background-color: #161b22; padding: 16px; border-radius: 6px; overflow-x: auto; font-family: 'SFMono-Regular', Consolas, 'Liberation Mono', Menlo, Courier, monospace;"><code>npm start</code></pre>
<p>이제 브라우저에서 <code>http://localhost:3000</code>으로 접속하여 Project-HSA를 시작할 수 있습니다.</p>
